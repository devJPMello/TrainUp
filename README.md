## Rodar o projeto

## Server
```bash
cd server
cp .env.example .env
npm i
docker compose up -d
npx drizzle-kit generate
npx drizzle-kit migrate
npm run seed
npm run server
```
Endpoints disponíveis [Swagger API Docs](http://localhost:3333/docs)

## Web
```bash
cd web
cp .env.local.example .env.local
npm i
npm run web
```
Acessar a [Aplicação Web](http://localhost:5173/login)

## Mobile
```bash
cd mobile
npm i
npm run mobile
```
